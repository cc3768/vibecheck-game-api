# Rewards System

- **Service name:** `rewards-system`
- **Default port:** `41740`
- **Role:** reward granting, validation, previews, history sources.

## Key Routes

- `POST /api/v1/rewards/grant`
- `POST /api/v1/rewards/preview`
- `GET /api/v1/rewards/history/:characterId`
- `POST /api/v1/rewards/validate`
- `POST /api/v1/rewards/from-quest`
- `POST /api/v1/rewards/from-npc`
