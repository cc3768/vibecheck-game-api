# XP System

- **Service name:** `xp-system`
- **Default port:** `41735`
- **Role:** xp evaluation, previewing, skill curves/rules.

## Key Routes

- `POST /api/v1/xp/evaluate`
- `POST /api/v1/xp/apply-direct`
- `POST /api/v1/xp/preview`
- `POST /api/v1/xp/from-actions`
- `GET /api/v1/xp/skill/:skillKey/rules`
- `GET /api/v1/xp/skill/:skillKey/curve`
